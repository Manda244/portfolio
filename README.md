# Portfolio

Site statique minimal pour présenter votre travail.

Ouvrir localement:

```sh
# 1) Ouvrir directement le fichier
# - double-cliquez sur index.html

# 2) Ou lancer un serveur local (recommandé)
python3 -m http.server 8000
# puis ouvrir http://localhost:8000
```

Personnalisation rapide:
- Remplacez "Votre Nom" dans `index.html` par votre nom.
- Mettez à jour les projets, descriptions et les liens GitHub.
- Remplacez `images/avatar.svg` par votre photo si vous le souhaitez.
